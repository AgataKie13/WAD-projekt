# WAD-projekt
Nazwa projektu:
Wielowymiarowa analiza danych dotyczących rezerwacji hotelowych ze zbioru "Hotel Booking"

Autorzy projektu: 
Agata Kierznikowicz (s190409), Zuzanna Łomża (s190274)

Opis projektu:
Wielowymiarowa analiza zachowań klientów hotelowych, identyfikacja czynników wpływających 
na anulowanie rezerwacji oraz wykrycie naturalnych grup rezerwacji, które różnią się między 
sobą pod względem charakterystyki gości i wzorców pobytu.
